Continue working with JavaScript

There are numerous directions to go with JavaScript from this point. Pick one of the following activities based on which sounds most interesting or useful to you. Each of these assumes you are loading a Carto map using HTML and JavaScript. You could continue with the page and map you have been using for the past few assignments, start a new one, or work on one that is more relevant to your project.

Your choices are:

(A) Add another layer to your map

Modify the map that you have made in JavaScript to have two or more layers. You can use this Glitch site as an example to work from.

(B) Filter data

Filter the data displayed on the map using SQL. Choose one or a few form inputs:

  • buttons (<button>) (Glitch example)
  • checkboxes (<input type="checkbox" />) (Glitch example)
  • text inputs (<input type="text" />) (Glitch example)
  • dropdowns (<select>) (Glitch example)

When the user changes the input(s), your page should filter the data on your map. This HTML Dog page gets into specifics of many HTML form elements.

(C) Add popups

Use JavaScript to create popups for your map that show the feature's data. Remember that one benefit of using JavaScript is you can embed videos and fully control the styles on your popups, so try to take advantage of that. Here's a Glitch example.
